Install BreakFreeDNS 3.0

BreakFreeDNS is a configuration profile, so install it by opening the .mobileconfig file directly in Safari. Do not use an itms-services://?action=download-manifest URL for this profile; that URL scheme is for OTA app manifests and will open iTunes without installing the DNS profile.

Download BreakFreeDNS 3.0 Profile
If iOS only downloads the file, open it from Files and then continue in Settings.
  1. Open this page in Safari on the iPhone or iPad.
  2. Tap the download button above and allow the profile download.
  3. Open Settings and tap Profile Downloaded.
  4. Tap Install, enter your passcode, and follow the prompts.
